# My First GitHub Repository

This repository was created as my first GitHub project while learning version control and GitHub fundamentals through the Programming Hero Batch 10 web development curriculum.

The main objective was to understand how GitHub repositories work and gain hands-on experience with basic Git operations.

- GitHub

- Uploading project files
- Making commits
- Basic version control concepts
- Managing source code online

- Creating and managing repositories
